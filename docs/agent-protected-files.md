# 代理保護檔案清單

## 目的
提供一份集中參考的清單，列出使用者目前指定為 Codex 不得修改的檔案，避免在自動化協作流程中發生誤改。

## 介面 / 參數
- `docs/agent-protected-files.md`：供所有代理或協作者查閱的最新受保護檔案來源。
- `AGENTS.md`：摘要說明此清單的存在與維護方式。

## 使用範例
- 在進行修改前，檢查此清單是否包含預計編輯的檔案；若在清單內，停止操作並與使用者確認。
- 若需要新增或移除檔案，請先取得使用者同意，再更新本檔案與 `AGENTS.md`。

## 測試要點
- 無自動化測試；以人工確認為主。每次更新後需再次檢視清單內容是否正確。

## 已知限制
- 清單需人工維護，可能與實際限制存在時間差。
- 僅規範 Codex 代理的操作；其他協作者仍需自行遵守使用者的指示。
