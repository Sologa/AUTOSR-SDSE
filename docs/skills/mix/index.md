## 統一使用流程（適用所有 skills）

> 若剛安裝新 skills，請先重啟 Codex 讓它載入。

1. **確認已載入**：在 Codex 內輸入 `/skills` 或輸入 `$` 查看清單，確認目標 skill 已出現。
2. **啟動 skill**：在對話中輸入 `$<skill-name>` 並附上你的任務描述。
3. **依 SKILL.md 操作**：每個 skill 的「使用條件、輸入格式、範例」都寫在該 skill 的 `SKILL.md`。
4. **提供必要資訊**：若 `SKILL.md` 要求 API key、檔案路徑、查詢條件或欄位格式，請在同一則訊息中一次給齊。
5. **遵守限制與驗證**：若 `SKILL.md` 有限制或驗證步驟，務必依序完成。
6. **調整與重試**：若輸出不符合預期，回到 `SKILL.md` 的範例與限制，補充缺失資訊後再次呼叫。
